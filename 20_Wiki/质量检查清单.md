# 质量检查清单

## 内容

- 核心结论是否清楚。
- 是否存在未经确认的事实。
- 是否有过时工具、价格、版本、许可信息。
- 术语是否首次解释。
- 结尾是否有明确收束。

## 画面

- 字幕在手机上可读。
- 关键信息没有被 UI 或字幕遮挡。
- 图表颜色和标签清楚。
- 动画没有只炫技而不解释。
- 镜头之间的视觉逻辑连续。
- 如果用 HTML 页面录屏，必须在实际录制屏幕尺寸下检查是否裁切。
- 16:9 HTML 画布建议增加窗口自适应缩放，避免小屏幕打开时内容显示到屏幕外。

## 声音

- 旁白音量稳定。
- BGM 不压过人声。
- 停顿足够让观众理解复杂概念。
- 专有名词读音统一。
- 若使用 MiniMax 配音，先在单页阶段出样音，全片文案锁定后再统一生成最终旁白。

## 导出

- 分辨率符合平台。
- 帧率统一。
- 文件名包含日期、项目名、版本号。
- 保留可编辑工程文件。
- 成片、字幕、封面分别归档。
