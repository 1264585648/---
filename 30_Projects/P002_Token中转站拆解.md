# P002 Token 中转站拆解

## 基本信息

- 项目 ID：P002
- 项目名：低价 AI API 背后：Token 中转站到底靠什么赚钱？
- 状态：制作中
- 目标受众：开发者、AI 创作者、对 AI 成本与合规感兴趣的观众
- 目标时长：8-10 分钟
- 画幅：16:9

## 一句话定位

用低价 API 广告作为入口，拆解 Token 中转站的定义、需求来源、赚钱逻辑、风险边界和未来形态。

## 核心观众问题

- 低价 AI API 为什么会存在？
- 所谓“官方低价 API”到底是不是官方？
- Token 中转站靠什么赚钱？
- 普通用户和企业真正承担什么风险？
- 未来灰色中转会不会消失？

## 当前脚本结构

1. 开场：低倍率、国内直连、官方 API，你刷到过吗？
2. 定义：Token 中转站是夹在用户和模型厂商之间的 API 网关。
3. 需求：Token 消耗从人机聊天变成机器流水线。
4. 赚钱：价格差、额度复用、计费规则、缓存复用。
5. 分层：正规代理、灰色套利、黑产来源。
6. 外壳：为什么它看起来像一个正规 API SaaS。
7. 风险：稳定性、数据、合规、资金。
8. 压缩：上游风控和价格战会削弱套利空间。
9. 未来：灰色中转被压缩，合规 AI 网关留下。
10. 结尾：便宜不是护城河，可信才是。
