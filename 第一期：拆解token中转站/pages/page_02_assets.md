# 第 2 页素材包：什么是中转站，以及为什么会催生中转站

## 素材目标

这一页要让观众同时理解两件事：

1. Token 中转站是什么。
2. 为什么 AI API 市场会催生出这样的中间层。

## 可上屏主标题

首选：

```text
什么是中转站，为什么会出现？
```

备选：

```text
它不是模型，而是转接层
```

```text
低价 API 背后的中间网关
```

## 可上屏副标题

```text
夹在用户和模型厂商之间，负责转发、包装和重新计费
```

## 关键词素材

建议出现：

- API 网关
- 转接层
- 统一接口
- 协议转换
- 流量转发
- 二次计费
- 成本压力
- 访问不稳
- 速率限制
- 多模型管理

避免出现：

- 具体账号池操作
- 绕过风控
- 盗刷
- 撞库
- 可复制接入路径

## 可视化元素

### 元素 1：三方网关结构图

```text
用户 / 开发者
        ↓
Token 中转站
        ↓
上游模型厂商
```

或者横向版本：

```text
用户 / 开发者 -> Token 中转站 -> 上游模型厂商
```

节点标签：

```text
DOWNSTREAM
GATEWAY
UPSTREAM
```

### 元素 2：中转动作标签

```text
统一接口
协议转换
流量转发
二次计费
```

### 元素 3：催生压力四卡片

```text
官方 API 成本高
海外访问不稳定
速率限制卡业务
多模型账单分散
```

### 元素 4：结论锁定卡

```text
中转站不是模型，
而是成本、访问和管理压力催生出的 API 中间层。
```
