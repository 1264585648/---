# 第 3 页素材包：三类中转站概览

## 素材目标

这一页用概览方式建立风险分层：不是所有中转站都一样，要先区分相对合规、灰色地带和违法高风险三类。

## 可上屏主标题

首选：

```text
三类中转站，不要混为一谈
```

备选：

```text
从合规网关，到灰色套利，再到违法来源
```

```text
低价 API 背后，有三种来源
```

## 可上屏副标题

```text
价格越低，来源越不透明，风险越高
```

## 关键词素材

建议出现：

- 相对合规
- 灰色地带
- 违法高风险
- API 中转
- 企业网关
- 订阅转 API
- 白嫖账号
- 凭证复用
- API Key 撞号
- 盗刷信用卡

避免出现：

- 具体网站或商家名
- 可复现步骤
- 工具名称
- 操作路径
- 交易入口

## 可视化元素

### 元素 1：风险光谱条

```text
相对合规  ->  灰色地带  ->  违法高风险
```

颜色建议：

- 相对合规：青绿色
- 灰色地带：琥珀色
- 违法高风险：红色

### 元素 2：三栏卡片

第一栏：

```text
01 相对合规 API 中转
授权采购
企业网关
账单审计
SLA
```

第二栏：

```text
02 订阅转 API 灰色地带
订阅额度包装
授权边界模糊
平台规则风险
```

第三栏：

```text
03 几乎零成本违法来源
白嫖账号
凭证复用
API Key 撞号
盗刷信用卡
```

### 元素 3：结论条

```text
先问来源，再谈价格
```

或者：

```text
价格越低，来源越不透明，风险越高
```

## 口播对位

| 口播段落 | 对应素材 |
| --- | --- |
| “要先把中转站分成三类” | 风险光谱条 |
| “第一类，是相对合规的 API 中转” | 第一栏卡片 |
| “第二类，是订阅转 API 的灰色地带” | 第二栏卡片 |
| “第三类，就不是灰色了” | 第三栏卡片 |
| “真正要问的是上游来源” | 结论条 |

## 不建议素材

- 真实账号池截图。
- 真实 API Key 样例。
- 真实盗刷或撞库流程。
- 真实支付页。
- 真实商家价格表。
