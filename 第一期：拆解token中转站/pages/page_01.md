# 第 1 页：开场钩子：你刷到过这种低价 API 广告吗？

## 本页要讲清楚的一句话

低倍率、国内直连、官方 API、注册送额度、低至 0.1 元兑换 1 美元额度，这类广告之所以有吸引力，是因为 Token 已经成为 AI 时代的基础消耗品；但价格低到离谱时，真正的问题不是“便宜不便宜”，而是“便宜从哪里来，风险被转移给了谁”。

## 内容概要

这一页用观众熟悉的广告语直接开场：低倍率、国内直连、官方 API、GPT / Claude / Gemini 全都有、注册即送、即开即用。

随后快速把问题从“薅便宜接口”拉回到视频主题：在 AI 应用里，无论写文案、读论文，还是接 API 做产品，本质都在消耗 Token。Token 就像算力世界里的电表读数，是模型能力被商品化之后最基础的计费单位。

最后抛出本期悬念：市面上的 API 中转平台越来越多，价格越来越低，所谓的“低价官方 API”到底是真的官方，还是披着官方外衣的二道贩子？那些便宜到离谱的 Token，背后藏着什么风险？

## 关键论点

- Token 是 AI 时代的基础计费单位。
- 低价 API 广告通常会把“低倍率、国内直连、官方 API、注册送额度、全模型支持”放在最显眼的位置。
- 极低价格本身就是一个需要追问的信号。
- 第一页不展开黑产细节，只建立问题意识：便宜从哪里来，风险由谁承担。
