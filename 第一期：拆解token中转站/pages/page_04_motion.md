## 视觉方向

这一页聚焦第三页中的第一类：相对合规的 API 中转。

- 左侧说明它是什么：企业级 AI 网关 / 正规代理。
- 右侧分三组：额度来源、服务价值、局限性。
- 局限性用琥珀色强调，不使用强烈红色，避免把相对合规类误判为黑产。

- 不能和第三页“三类概览”重复。
