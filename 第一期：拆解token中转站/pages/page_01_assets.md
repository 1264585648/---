# 第 1 页素材包：开场钩子：你刷到过这种低价 API 广告吗？

## 素材目标

这一页的素材要服务一个目的：

让观众在前 10 到 20 秒内意识到，低价 API 广告不是普通促销，而是一个值得追问的灰色转移链路。

## 可上屏主标题

首选：

```text
这种低价 API，你刷到过吗？
```

备选：

```text
低倍率、国内直连、官方 API，真的有这么便宜？
```

```text
Token 时代，便宜到离谱的 API 到底靠什么赚钱？
```

## 可上屏副标题

```text
Token 是 AI 时代的基础计费单位，像算力世界里的电表读数
```

## 关键视觉元素

### 元素 1：低价广告流

用途：

模拟开发者社群、广告投放页、渠道页里常见的低价 API 宣传语。

可用文案：

```text
国内直连 · 低延迟
官方 API · 稳定转发
GPT / Claude / Gemini 全都有
低至 0.1 元 / $1 额度
注册即送 · 即开即用
```

注意：

- 不出现真实商家名称。
- 不出现真实购买入口。
- 不出现可执行交易信息。

### 元素 2：问题钩子卡片

用途：

把观众从“便宜”拉回到“为什么便宜”。

可用结构：

```text
你是否经常看到这样的广告？
```

### 元素 3：Token 计量表

用途：

把抽象的 Token 概念钉在“基础计费单位”上。

可用标签：

```text
TOKEN
BASIC UNIT
```

### 元素 4：价格差问题块

用途：

把“官方 API”和“低价中转”之间的差异抽象出来。

可用结构：

```text
官方 API：按 Token 计费，有速率限制和合规边界
低价中转：更便宜、更宽松，但说明往往更少

如果价格低到离谱，风险被转移给了谁？
```

## 可用类比

首选类比：

```text
Token 像 AI 时代的电表读数，每一次调用都在耗电
```

谨慎使用类比：

```text
中转站像算力二道贩子
```

这个说法直观，但略带判断色彩。可用于标题或转场，不建议在第一页反复强调。

## 不建议使用的素材

- 真实中转站后台截图。
- 真实商家价格表。
- 真实 API Key、账户池、支付页面。
- 具体规避风控的流程图。
- 盗刷、撞库、洗钱的操作细节。
