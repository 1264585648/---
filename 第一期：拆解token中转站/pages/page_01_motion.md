# 第 1 页动效说明：开场钩子

## 页面文件

- HTML 动效页：`renders/html/page_01_opening.html`

## 画布规格

- 尺寸：1600 x 900
- 比例：16:9
- 形式：单文件 HTML + CSS 动画
- 背景：深色技术调查风格，带网格和扫描线

## 视觉方向

强化“广告刷屏 + 问题追问 + Token 基础计费单位”三层关系：

- 右侧先出现低价 API 广告卡片。
- 中段弹出“你是否经常看到这样的广告？”。
- 左下再收束到 Token 计量表和基础单位说明。

## 动效时间线

| 时间 | 内容 |
| --- | --- |
| 0.18s | 风险标签 `LOW PRICE API SIGNAL` 出现 |
| 0.55s | 主标题出现：`这种低价 API，你刷到过吗？` |
| 1.05s | 副标题出现，拉出“便宜从哪里来”的问题 |
| 1.45s - 2.53s | 广告卡片依次进入 |
| 3.35s | 右侧问题卡片出现 |
| 4.55s | 左下标签条出现 |
| 5.65s | Token 计量表和基础单位说明出现 |
| 7.35s | 价格差问题块出现，转入正式分析 |

## 当前检查结果

- 主标题没有越界。
- 广告卡片没有互相遮挡。
- 关键文案均位于主内容区内。
- 文案长度适合 16:9 画面展示。
