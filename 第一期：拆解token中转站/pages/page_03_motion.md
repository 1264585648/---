# 第 3 页动效说明：三类中转站概览

## 页面文件

- HTML 动效页：`renders/html/page_03_workflow.html`
- Markdown 页面：`pages/page_03.md`
- 素材包：`pages/page_03_assets.md`

## 画布规格

- 尺寸：1600 x 900
- 比例：16:9
- 形式：单页 HTML + CSS 动效
- 风格：深色技术调查、风险光谱、三栏概览

## 视觉方向

这一页不是继续解释调用量，而是建立分类框架。

- 左侧给出主判断：三类中转站，不要混为一谈。
- 右侧用三栏卡片展示：相对合规、灰色地带、违法高风险。
- 顶部或中部用风险光谱条表达从低风险到高风险的迁移。
- 红色只用于第三类，避免全页过度警报化。

## 动效时间线

| 时间 | 内容 |
| --- | --- |
| 0.20s | 页面底板和章节标签出现 |
| 0.55s | 主标题出现 |

## 当前质检目标

- 三类卡片必须一眼能区分。
- 第三类可以点名风险类型，但不出现操作流程。
- 不出现真实平台、真实账号、真实 API Key 或支付信息。
- 文案不能过密。
