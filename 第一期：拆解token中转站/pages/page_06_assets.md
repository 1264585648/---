# 第 6 页素材包：第三类，几乎零成本的违法来源

## 素材目标

这一页要把第三类中转站讲成重点风险：它不是低价采购，而是把非法账号、凭证、额度和支付能力包装成低价 API。

## 可上屏主标题

首选：

```text
第三类：几乎零成本的违法来源
```

备选：

```text
便宜到零成本，往往不是技术红利
```

```text
低价 API 最危险的一层
```

## 可上屏副标题

```text
账号、凭证、额度和支付能力被非法转卖
```

## 关键词素材

建议出现：

- 批量注册账号
- 虚拟卡白嫖订阅
- 撞号
- 撞 API Key
- 网页转 API
- 凭证复用
- 非法额度
- 脏额度
- 断供
- 法律风险

避免出现：

- 具体工具名
- 注册机流程
- 接码平台
- 撞库脚本
- 盗刷步骤
- 网页转 API 实现方式

## 可视化元素

### 元素 1：五类高风险来源卡

```text
批量注册账号
虚拟卡 / 黑卡白嫖订阅
撞号 / 撞 API Key
网页能力转 API
凭证复用与转卖
```

### 元素 2：成本转嫁链路

```text
非法来源 -> 低价 API -> 下游用户 -> 封号 / 断供 / 数据风险 / 法律风险
```

### 元素 3：警示结论

```text
越接近零成本，
越可能是违法成本外包。
```

## 不建议素材

- 真实账号、API Key、Cookie、Token。
- 真实盗刷或撞库案例截图。
- 可复现流程图。
- 工具名和教程式界面。
