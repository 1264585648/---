# 视频制作知识库

这是一个参考 Karpathy `llm-wiki` 思路搭建的本地视频制作知识库。

核心原则：
- `10_Sources` 保存原始材料和引用记录，尽量不改写。
- `20_Wiki` 保存被 AI 编译、重组、可持续维护的知识页。
- `30_Projects` 保存每次视频制作的完整过程和复盘。
- `40_StyleMemory` 保存从会话和项目中归纳出的个人风格倾向。
- `90_System` 保存知识库维护协议、标签、字段和更新流程。

推荐工作流：
1. 把新材料、会话导出、网页摘录、灵感草稿放进 `00_Inbox`。
2. 由 AI 读取 `90_System/知识库维护协议.md`，把材料登记到 `10_Sources/来源索引.md`。
3. AI 将可靠内容编译到 `20_Wiki`，只在 wiki 页里沉淀结论。
4. 每个视频项目从 `50_Templates/项目卡模板.md` 复制一份到 `30_Projects`。
5. 项目结束后更新 `40_StyleMemory/风格画像.md` 和 `40_StyleMemory/制作习惯日志.md`。

当前首批材料：
- `CLI科普视频制作工具推荐.md`
- `内容生成完整视频路径.md`

注意：如果在 PowerShell 中直接读取中文 Markdown 出现乱码，优先用支持 UTF-8 的编辑器打开；不要基于乱码内容做摘要。
